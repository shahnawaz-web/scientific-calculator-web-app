# scientific-calculator-web-app
A powerful and user-friendly scientific calculator built using HTML, CSS, and JavaScript. This application performs basic arithmetic operations as well as advanced mathematical functions such as trigonometry, logarithms, exponentiation, and more.

 # Features:
- Basic operations: Addition, Subtraction, Multiplication, Division
- Advanced functions: sin, cos, tan, log, ln, square root, power(x*2,x*3,x*y),n!,e*x
- Responsive design (mobile-first)
- Clean and modern UI
- Real-time calculation display
- Error handling for invalid inputs

# Tech Stack:
- HTML5
- CSS3
- JavaScript (ES6)
