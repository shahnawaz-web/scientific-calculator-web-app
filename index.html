About This Scientific Calculator

This free online scientific calculator supports a full range of mathematical operations. The dual-display interface shows your result prominently at the top and your current input expression below, so you always know both what you entered and what the answer is.

How to Use

  • Type numbers using digit buttons or keyboard (0–9)
  • Press = or Enter to compute the result
  • Use Clear to clear all, or Back to delete last character
  • Toggle DEG / RAD for trigonometric function mode
  • Memory: M+ adds to memory, M− subtracts, MR recalls, MC clears
  • Tap any history item to restore that result

Supported Functions

√ (square root), eˣ (exponential), sin / cos / tan, sinh / cosh / tanh (hyperbolic), 1/x (reciprocal), log (base-10), n! (factorial), x² (square), x³ (cube), xʸ (power), +/− (sign toggle), % (percentage).
